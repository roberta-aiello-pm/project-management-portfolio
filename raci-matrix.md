# RACI Matrix

## Project: Online Booking System Implementation

---

## RACI Definition

R = Responsible (performs the task)  
A = Accountable (owns the task)  
C = Consulted (provides input)  
I = Informed (kept informed)

---

## Roles

- Project Sponsor
- Project Manager
- Project Coordinator
- Technical Team
- Stakeholders
