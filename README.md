## Project 2 — Sauce & Spoon: Tablet Integration Pilot
**Context:** Academic Project – Google PM Certificate
**Methodology:** Agile
**Scope:** Technology upgrade for a restaurant chain — implementation of tabletop ordering tablets to modernize guest experience, streamline order efficiency and increase digital marketing enrollment.

### Key Results Achieved
- ✅ **72% guest satisfaction rate** validated tablet system viability
- ✅ **78% newsletter conversion rate** through in-tablet data capture
- ✅ **56% of food orders** delivered in under 20 minutes
- ✅ **100% of test participants** used tablets for main course ordering
- ✅ **76% staff adoption rate** through guided onboarding

| Deliverable | Description |
