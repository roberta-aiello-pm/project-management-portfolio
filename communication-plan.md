# Communication Plan

## Project: Online Booking System Implementation

---

## Communication Objectives

The objective of this communication plan is to ensure all stakeholders receive the right information at the right time.

---

## Stakeholder Communication Matrix

| Stakeholder | Information | Frequency | Method | Owner |
|------------|-------------|-----------|--------|-------|
| Project Sponsor | Project status report | Weekly | Email | Project Coordinator |
| Project Team | Tasks and updates | Daily | Meeting | Project Coordinator |
| Stakeholders | Progress update | Bi-weekly | Presentation | Project Manager |
| Technical Team | Technical updates | Weekly | Meeting | Project Coordinator |
